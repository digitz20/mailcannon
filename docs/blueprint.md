# **App Name**: MailCannon

## Core Features:

- Email List Input: Input a list of recipient email addresses.
- Email Composition: Compose the email content with a subject and body.
- Batch Email Sending: Send the composed email to all recipients in the list.
- Attachment Handling: Option to attach a file or document to the email.

## Style Guidelines:

- Primary color: Clean white or light gray for the background.
- Secondary color: Dark gray or black for text to ensure readability.
- Accent: Blue (#3498db) for primary actions (e.g., send button).
- Clean, organized layout with clear sections for input and composition.
- Use a clear, readable font (e.g., Open Sans).
- Simple, intuitive icons for actions like adding attachments or sending emails.
